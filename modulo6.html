Bem-vindo ao Módulo 6: Seletores CSS

Vamos aprender a selecionar e estilizar elementos com CSS de maneira eficiente!

O que são Seletores CSS?

Os **seletores CSS** são usados para selecionar e aplicar estilos a elementos HTML específicos em uma página web. Eles são a base da estilização, permitindo que você defina quais elementos devem ser afetados por determinadas regras de estilo.

A escolha correta de seletores é fundamental para uma boa organização do código CSS e para garantir que o estilo seja aplicado corretamente.

1. Seletores de Tipo

Os **seletores de tipo** selecionam todos os elementos de um determinado tipo ou tag HTML. Por exemplo, para selecionar todos os parágrafos ou títulos, usamos o nome da tag como seletor.


                p {
                    color: red;
                }
                h1 {
                    font-size: 36px;
                }

O código acima seleciona todos os elementos <p> e <h1> e aplica os estilos definidos a eles. Todos os parágrafos terão a cor vermelha e os títulos <h1> terão tamanho de fonte 36px.

2. Seletores de Classe e ID

Os **seletores de classe** e **ID** permitem selecionar elementos específicos da página, sendo muito úteis para aplicar estilos de maneira mais granular.

Seletores de Classe

Os seletores de classe são precedidos por um ponto (.) e selecionam todos os elementos que possuem uma determinada classe.


                .meu-paragrafo {
                    font-size: 18px;
                    color: blue;
                }

Para aplicar o estilo, você adicionaria a classe ao elemento HTML da seguinte forma:


                <p class="meu-paragrafo">Este é um parágrafo estilizado.</p>

Seletores de ID

Os seletores de ID são precedidos por um hash (#) e são usados para selecionar um único elemento da página com o ID especificado. O ID deve ser único dentro de uma página.


                #titulo {
                    color: green;
                }

Para aplicar o estilo, você usaria o ID no elemento HTML:


                <h1 id="titulo">Este é o título estilizado.</h1>

3. Seletores Universais e Combinados

O **seletor universal** (<*>) seleciona todos os elementos da página. Já os **seletores combinados** permitem selecionar elementos que estão em relação uns com os outros.

Seletor Universal

O seletor universal seleciona todos os elementos da página e pode ser útil para aplicar um estilo geral a toda a página.


                * {
                    margin: 0;
                    padding: 0;
                }

Este código remove todas as margens e preenchimentos de todos os elementos.

Seletores Combinados

Você pode combinar seletores para aplicar estilos a elementos com base em sua posição ou relação com outros elementos. Alguns exemplos incluem:

  • elemento1 elemento2: Seleciona elemento2 que está dentro de elemento1.
  • elemento1 > elemento2: Seleciona elemento2 que é filho direto de elemento1.
  • elemento1 + elemento2: Seleciona o primeiro elemento2 que segue imediatamente elemento1.
  • elemento1 ~ elemento2: Seleciona todos os elemento2 que são irmãos de elemento1.

4. Seletores de Atributo

Os **seletores de atributo** permitem selecionar elementos com base no valor de um atributo específico. Isso é útil quando você precisa selecionar elementos com atributos específicos, como href, src, etc.


                a[href^="https"] {
                    color: green;
                }

Este exemplo seleciona todos os links <a> cujo atributo href começa com https e aplica a cor verde a eles.

5. Seletores Pseudo-classes e Pseudo-elementos

As **pseudo-classes** e **pseudo-elementos** permitem selecionar elementos em estados específicos ou partes de um elemento, sem a necessidade de adicionar classes ou IDs no HTML.

Pseudo-classes

As pseudo-classes são usadas para selecionar elementos com base em seu estado. Alguns exemplos incluem:

  • :hover: Seleciona um elemento quando o mouse passa sobre ele.
  • :focus: Seleciona um elemento que está em foco.
  • :active: Seleciona um elemento que está sendo clicado.

                a:hover {
                    color: red;
                }

Este exemplo altera a cor do link para vermelho quando o mouse passa sobre ele.

Pseudo-elementos

Os pseudo-elementos são usados para selecionar partes de um elemento, como o início ou o fim de um texto. Alguns exemplos incluem:

  • ::before: Insere conteúdo antes de um elemento.
  • ::after: Insere conteúdo após um elemento.

                p::before {
                    content: "Nota: ";
                    font-weight: bold;
                }

Este exemplo adiciona o texto "Nota: " antes de todo parágrafo.

6. Seletores de Descendentes, Filhos e Irmãos

Os seletores de **descendentes**, **filhos** e **irmãos** são usados para selecionar elementos com base em sua relação hierárquica no DOM.

Seletores de Descendentes

Seleciona um elemento que é descendente de outro. Por exemplo:


                div p {
                    color: blue;
                }

Isso seleciona todos os <p> dentro de um <div> e aplica a cor azul ao texto.

Seletores de Filhos

Seleciona um elemento que é filho direto de outro.


                div > p {
                    color: green;
                }

Seletores de Irmãos

Seleciona um elemento que é irmão de outro. Por exemplo, o próximo <p> após um <h1> pode ser selecionado assim:


                h1 + p {
                    color: purple;
                }
