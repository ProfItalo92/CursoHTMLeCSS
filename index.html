Bem-vindo ao Módulo 1: Introdução ao HTML

Vamos aprender os conceitos básicos de HTML!

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação usada para criar páginas na web. Ele é composto por uma série de elementos chamados "tags", que são usados para estruturar o conteúdo da página.

Estrutura Básica de um Documento HTML

Todo documento HTML começa com a declaração do tipo de documento e a tag <html>, seguida pelas tags <head> e <body>.


                <DOCTYPE html>
                <html lang="pt-br">
                <head>
                   <meta charset="UTF-8">
                   <meta name="viewport" content="width=device-width, initial-scale=1.0">
                   <title> Minha Primeira Página </title>
                </head>
                <body>
                    <h1>Bem-vindo ao HTML </h1>
                    <p> Este é um parágrafo de exemplo. </p>
                </body >
                </html>

Principais Tags HTML

  • <html>: Define o documento HTML.
  • <head>: Contém metadados sobre o documento (como título e links para CSS).
  • <body>: Contém o conteúdo visível da página.
  • <h1> a <h6>: Títulos de diferentes níveis.
  • <p>: Define um parágrafo.

Exemplo Prático


                <html>
                    <head>
                        <title>Exemplo de Página</title>
                    </head>
                    <body>
                        <h1>Meu Primeiro Título</h1>
                        <p>Este é um parágrafo simples.</p>
                    </body>
                </html>

Conclusão

HTML é a base da criação de páginas web, permitindo estruturar o conteúdo de forma organizada e compreensível para os navegadores.
