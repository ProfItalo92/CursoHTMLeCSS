Bem-vindo ao Módulo 5: Introdução ao CSS

Vamos aprender como estilizar páginas HTML com CSS!

O que é CSS?

CSS (Cascading Style Sheets) é uma linguagem de estilização usada para definir a aparência de uma página web escrita em HTML. Enquanto o HTML é responsável pela estrutura e conteúdo da página, o CSS é responsável pelo estilo, como cores, fontes, espaçamento e layout.

O CSS permite separar a estrutura da apresentação, o que torna o código mais organizado, reutilizável e fácil de manter.

1. Como Funciona o CSS?

Existem três maneiras principais de aplicar CSS a uma página web:

  • CSS Inline: Aplica o estilo diretamente dentro de uma tag HTML, usando o atributo style.
  • CSS Interno: Aplica o estilo dentro de uma tag <style> no cabeçalho do documento HTML.
  • CSS Externo: Aplica o estilo em um arquivo separado, linkado à página HTML usando a tag <link>.

1.1. CSS Inline

O CSS inline é aplicado diretamente em um elemento HTML através do atributo style.


                <p style="color: blue; font-size: 20px;">Este texto é azul e tem fonte de 20px.</p>

O exemplo acima aplica um estilo diretamente ao parágrafo, tornando o texto azul e com uma fonte de 20px.

1.2. CSS Interno

O CSS interno é definido dentro da tag <style> no cabeçalho do documento HTML. Ele afeta todos os elementos que correspondem aos seletores.


                <head>
                    <style>
                        p {
                            color: red;
                            font-size: 18px;
                        }
                    </style>
                </head>
                <p>Este texto será vermelho e com fonte de 18px.</p>

1.3. CSS Externo

O CSS externo é o mais comum e recomendado. Ele é colocado em um arquivo separado, com a extensão .css, e é linkado à página HTML com a tag <link>.


                <link rel="stylesheet" href="estilos.css">

O conteúdo do arquivo estilos.css seria:


                p {
                    color: green;
                    font-size: 16px;
                }

2. Seletores CSS

Os seletores são usados para "selecionar" os elementos HTML que serão estilizados. Existem diferentes tipos de seletores:

  • Seletores de tipo: Seleciona todos os elementos de um tipo específico, como p, h1, div.
  • Seletores de classe: Seleciona todos os elementos que têm uma classe específica, usando o ponto antes do nome da classe (.classe).
  • Seletores de ID: Seleciona um único elemento com um ID específico, usando o caractere de hash (#id).
  • Seletores de atributo: Seleciona elementos que possuem um atributo específico.

2.1. Seletores de Tipo


                p {
                    color: orange;
                }

O exemplo acima estiliza todos os parágrafos (p) da página, tornando o texto laranja.

2.2. Seletores de Classe


                .meuTexto {
                    color: purple;
                    font-size: 22px;
                }

Para usar este seletor, você aplicaria a classe meuTexto em um elemento HTML, como mostrado abaixo:


                <p class="meuTexto">Este texto será roxo e com fonte de 22px.</p>

2.3. Seletores de ID


                #meuParagrafo {
                    color: red;
                    font-weight: bold;
                }

Usando o seletor de ID, você pode estilizar um elemento único na página. No exemplo abaixo:


                <p id="meuParagrafo">Este parágrafo será vermelho e em negrito.</p>

3. Propriedades de CSS

O CSS oferece várias propriedades que permitem alterar a aparência de elementos HTML. Vamos ver algumas das propriedades mais comuns:

  • color: Define a cor do texto.
  • font-size: Define o tamanho da fonte.
  • background-color: Define a cor de fundo.
  • margin: Define a margem ao redor de um elemento.
  • padding: Define o preenchimento dentro de um elemento.

3.1. Cores e Fontes


                p {
                    color: blue;
                    font-size: 18px;
                }

Este exemplo altera a cor do texto para azul e o tamanho da fonte para 18px.

3.2. Margens e Preenchimento


                div {
                    margin: 20px;
                    padding: 15px;
                }

A propriedade margin define o espaço ao redor de um elemento, enquanto a propriedade padding define o espaço dentro do elemento.

4. Layouts com CSS

O CSS oferece várias formas de criar layouts. Vamos explorar o **Flexbox** e o **Grid Layout**, dois dos métodos mais populares para construir layouts responsivos e dinâmicos.

4.1. Flexbox

O **Flexbox** é um modelo de layout unidimensional que permite distribuir o espaço entre os elementos de forma eficiente. Ele é usado quando você precisa alinhar itens em uma linha ou coluna.


                .container {
                    display: flex;
                    justify-content: space-between;
                }
                .item {
                    width: 30%;
                }

4.2. Grid Layout

O **Grid Layout** permite criar layouts bidimensionais, organizando os elementos em linhas e colunas.


                .grid {
                    display: grid;
                    grid-template-columns: repeat(3, 1fr);
                    gap: 20px;
                }
                .item {
                    background-color: lightblue;
                }

5. Responsividade e Media Queries

**Media queries** permitem que o CSS se adapte a diferentes tamanhos de tela, tornando o site responsivo.


                @media (max-width: 768px) {
                    body {
                        font-size: 14px;
                    }
                }

O exemplo acima diminui o tamanho da fonte para 14px em telas menores que 768px.

6. Transições e Animações em CSS

O CSS permite criar **transições** e **animações** para adicionar interatividade à sua página.

6.1. Transições


                .botao {
                    transition: background-color 0.3s ease;
                }
                .botao:hover {
                    background-color: #4CAF50;
                }

A propriedade transition permite que a cor de fundo do botão mude suavemente quando o mouse passa sobre ele.

6.2. Animações


                @keyframes animacao {
                    from {
                        opacity: 0;
                    }
                    to {
                        opacity: 1;
                    }
                }

                .elemento {
                    animation: animacao 2s ease-in-out;
                }

A propriedade animation permite criar animações complexas no CSS.
