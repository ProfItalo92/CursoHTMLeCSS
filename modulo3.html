Vamos aprender sobre as tags de listas e tabelas no HTML!

Listas HTML

As **listas** em HTML são utilizadas para organizar dados em grupos. Existem três tipos principais de listas:

  • Listas não ordenadas: Usadas para itens que não precisam estar em uma ordem específica.
  • Listas ordenadas: Usadas quando a ordem dos itens é importante, geralmente numerada.
  • Listas de definição: Usadas para exibir pares de termos e descrições.

1. Listas Não Ordenadas

A tag <ul> é usada para criar uma lista não ordenada, e dentro dela são adicionados os itens com a tag <li>.


                <ul>
                    <li>Item 1</li>
                    <li>Item 2</li>
                    <li>Item 3</li>
                </ul>

O exemplo acima cria uma lista com três itens, onde cada item é representado pela tag <li>.

2. Listas Ordenadas

A tag <ol> é usada para criar uma lista ordenada. Assim como nas listas não ordenadas, a tag <li> é usada para definir cada item da lista.


                <ol>
                    <li>Primeiro item</li>
                    <li>Segundo item</li>
                    <li>Terceiro item</li>
                </ol>

O exemplo acima cria uma lista numerada com três itens.

3. Listas de Definição

A tag <dl> é usada para criar listas de definição. Dentro dela, usamos as tags <dt> para definir o termo e <dd> para definir a descrição.


                <dl>
                    <dt>HTML</dt>
                    <dd>Uma linguagem de marcação usada para criar páginas web</dd>
                    <dt>CSS</dt>
                    <dd>Uma linguagem usada para estilizar páginas web</dd>
                </dl>

O exemplo acima cria uma lista de definição com dois termos e suas respectivas descrições.

Tabelas HTML

As **tabelas** são usadas para organizar dados em linhas e colunas. O HTML oferece a tag <table> para definir uma tabela, com as tags <tr> para as linhas, <td> para as células de dados e <th> para os cabeçalhos das colunas.

1. Criando uma Tabela

Para criar uma tabela básica, usamos a tag <table>, onde as linhas da tabela são definidas pela tag <tr>, e dentro de cada linha, as células são definidas com as tags <td> (dados) ou <th> (cabeçalho).


                <table>
                    <tr>
                        <th>Nome</th>
                        <th>Idade</th>
                    </tr>
                    <tr>
                        <td>João</td>
                        <td>25</td>
                    </tr>
                    <tr>
                        <td>Maria</td>
                        <td>30</td>
                    </tr>
                </table>

O exemplo acima cria uma tabela simples com duas colunas (Nome e Idade) e duas linhas de dados.

2. Atributos de Tabelas

As tabelas também podem ter atributos, como border para definir uma borda, cellspacing para definir o espaçamento entre as células, e cellpadding para definir o preenchimento dentro das células.


                <table border="1" cellspacing="2" cellpadding="5">
                    <tr>
                        <th>Nome</th>
                        <th>Idade</th>
                    </tr>
                    <tr>
                        <td>João</td>
                        <td>25</td>
                    </tr>
                </table>

Este exemplo cria uma tabela com borda, espaçamento entre as células e preenchimento dentro das células.
