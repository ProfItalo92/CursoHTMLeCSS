Bem-vindo ao Módulo 2: Tags e Atributos HTML

Vamos aprender sobre as principais tags e atributos do HTML!

O que são Tags HTML?

As **tags HTML** são os elementos usados para definir a estrutura de uma página web. Elas são usadas para criar e organizar o conteúdo da página, como textos, imagens, links, listas, entre outros. Cada tag tem um propósito específico e pode conter atributos que controlam seu comportamento ou aparência.

1. Tags de Estruturação de Página

A estrutura básica de um documento HTML começa com a tag <html>, que define o início de um documento HTML. Dentro da tag <html>, temos as tags <head> e <body>.


                <!DOCTYPE html>
                <html lang="pt-br">
                <head>
                    <meta charset="UTF-8">
                    <meta name="viewport" content="width=device-width, initial-scale=1.0">
                    <title>Exemplo de Página</title>
                </head>
                <body>
                    <h1>Bem-vindo ao HTML</h1>
                        <p>Este é um exemplo simples de uma página HTML.</p>
                </body>
                </html>

2. Tags de Texto e Formatação

O HTML oferece várias tags para estruturar o conteúdo de texto e aplicá-lo a diferentes tipos de formatação.

  • <h1> a <h6>: Define os títulos de diferentes níveis, sendo <h1> o mais importante e <h6> o menos importante.
  • <p>: Define um parágrafo de texto.
  • <strong>: Define texto em negrito.
  • <em>: Define texto em itálico.

                <h1>Título Principal</h1>
                <p>Este é um parágrafo com <strong>negrito</strong> e <em>itálico</em>.</p>

3. Tags de Listas

O HTML possui duas principais tags de listas:

  • <ul>: Cria uma lista não ordenada (com marcadores).
  • <ol>: Cria uma lista ordenada (numerada).

                <ul>
                    <li>Primeiro item</li>
                    <li>Segundo item</li>
                </ul>

                <ol>
                    <li>Primeiro item numerado</li>
                    <li>Segundo item numerado</li>
                </ol>

4. Tags de Links

A tag <a> é usada para criar links em uma página HTML. O atributo href especifica o destino do link.


                Clique aqui para visitar nosso site

5. Tags de Imagens

A tag <img> é usada para exibir imagens na página. O atributo src especifica o caminho da imagem, e o atributo alt fornece uma descrição alternativa da imagem.

6. Tags de Tabelas

O HTML também permite criar tabelas com a tag <table>, junto com as tags <tr> (para linhas), <td> (para células de dados) e <th> (para células de cabeçalho).


                <table>
                    <tr>
                        <th>Nome</th>
                        <th>Idade</th>
                    </tr>
                    <tr>
                        <td>João</td>
                        <td>25</td>
                    </tr>
                    <tr>
                        <td>Ana</td>
                        <td>30</td>
                    </tr>
                </table>

7. Tags de Formulários

Os formulários são essenciais para capturar informações do usuário. As tags mais comuns incluem <form>, <input>, <textarea> e <button>.


                <form action="/enviar" method="POST">
                    <label for="nome">Nome:</label>
                    <input type="text" id="nome" name="nome">
                    <button type="submit">Enviar</button>
                </form>

8. Atributos HTML

Os atributos em HTML são usados para fornecer informações adicionais sobre um elemento. Alguns atributos comuns incluem:

  • id: Identificador único para um elemento.
  • class: Define uma classe para aplicar estilos CSS.
  • style: Aplica estilos diretamente ao elemento.
  • href: Define o destino de um link.
