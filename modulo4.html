Bem-vindo ao Módulo 4: Links e Imagens HTML

Vamos aprender como adicionar links e imagens nas páginas HTML!

O que são Links e Imagens em HTML?

**Links** e **imagens** são dois dos elementos mais essenciais para criar páginas web interativas e ricas em conteúdo visual. Os links permitem que você navegue entre diferentes páginas ou sites, enquanto as imagens ajudam a ilustrar e a tornar o conteúdo mais visualmente atraente.

1. Como Criar Links em HTML

Os **links** em HTML são criados com a tag <a>, que é usada para definir um hyperlink. O principal atributo dessa tag é o href, que especifica o destino do link.


                <a href="https://www.exemplo.com">Clique aqui para visitar nosso site</a>

O código acima cria um link que, ao ser clicado, levará o usuário para o site https://www.exemplo.com.

1.1. Atributo href

O atributo href (Hypertext Reference) é utilizado para definir o destino do link. Ele pode apontar para:

  • Links Externos: Para acessar outros sites na web, basta passar a URL completa no atributo href.
  • Links Internos: Links que direcionam para outra parte da mesma página ou para outra página do mesmo site.

                <a href="https://www.exemplo.com">Acesse nosso site</a>
                <a href="#secao1">Ir para a Seção 1</a>

O primeiro link leva para uma página externa, enquanto o segundo link leva para uma seção interna da mesma página (supondo que haja um elemento com o id="secao1").

1.2. Links de Email e Telefones

Você também pode usar a tag <a> para criar links de **email** ou **telefone**:


                <a href="mailto:exemplo@dominio.com">Envie um e-mail</a>
                <a href="tel:+551199999999">Ligue para nós</a>

O link de email permite que o usuário abra o cliente de email para enviar uma mensagem, e o link de telefone permite que o usuário ligue para o número especificado diretamente de dispositivos móveis.

2. Como Inserir Imagens em HTML

As **imagens** em HTML são inseridas com a tag <img>. Essa tag não possui uma tag de fechamento, pois é uma tag auto-fechante. O atributo mais importante é src, que define o caminho da imagem, e alt, que fornece um texto alternativo caso a imagem não seja carregada.


                <img src="imagem.jpg" alt="Descrição da imagem">

O código acima carrega a imagem imagem.jpg e exibe a descrição "Descrição da imagem" caso a imagem não possa ser exibida.

2.1. Atributo src e alt

O atributo src especifica o caminho do arquivo de imagem. Esse caminho pode ser um arquivo local ou uma URL de uma imagem hospedada na web. O atributo alt é usado para melhorar a acessibilidade da página, proporcionando uma descrição textual da imagem para leitores de tela.


                <img src="logo.png" alt="Logotipo da Empresa">

2.2. Imagens Responsivas

Para tornar as imagens responsivas (adaptáveis ao tamanho da tela), você pode usar o CSS para controlar a largura e a altura das imagens.


                <img src="imagem.jpg" alt="Imagem Responsiva" style="width: 100%; height: auto;">

Este exemplo faz com que a imagem ocupe 100% da largura do seu elemento pai, mantendo a proporção da imagem.

3. Links e Imagens de Forma Avançada

Além de links simples e imagens estáticas, podemos usar imagens como links e até criar links de download para arquivos.

3.1. Imagens Como Links

Você pode usar uma imagem como link, clicando na imagem para redirecionar o usuário.


                <a href="https://www.exemplo.com">
                    <img src="imagem.jpg" alt="Imagem como link">
                </a>

3.2. Links de Download

Você também pode criar links para permitir que o usuário faça o download de arquivos, como documentos ou imagens.


                <a href="documento.pdf" download>Clique aqui para baixar o PDF</a>

O atributo download permite que o link seja tratado como um download em vez de um link para visualização.

Conclusão
